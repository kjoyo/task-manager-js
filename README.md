# task-manager-js
Simple task manager web application built with vanilla JavaScript.

## Project approach
This project was developed following an iterative workflow inspired by Scrum:
- Small incremental features
- Prioritized backlog
- Short development iterations

## Sprint 1
**Goal:** Basic task management functionality.

**Completed features:**
- Create tasks
- Validate empty input
- Mark tasks as completed
  
Sprint 1 completed successfully with all planned features delivered.

## Tech stack
- HTML
- CSS
- JavaScript
